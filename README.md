Projetos de Estudos em SQL
Este repositório reúne alguns projetos de estudos em SQL que desenvolvi para praticar consultas, modelagem de dados e análise exploratória. Cada pasta representa um mini‑projeto com foco em um contexto diferente de negócio.

Estrutura do repositório
Analise_loja_Musica/
Análise de uma base de dados de uma loja de música (estilo iTunes/Spotify), com foco em:

Identificação de clientes mais valiosos e países com maior faturamento.

Artistas, gêneros e álbuns mais vendidos.

Ticket médio por compra e comportamento de vendas por período.

Analise_northwind/
Uso do clássico banco Northwind para praticar:

Junções entre múltiplas tabelas (pedidos, clientes, produtos, fornecedores).

Métricas de vendas por região, categoria e funcionário.

Cálculo de faturamento, descontos, frete e análise de performance comercial.

Analise_vendas_ecommerce/
Análise de uma base de e‑commerce com foco em:

Indicadores de vendas (receita, número de pedidos, quantidade vendida).

Produtos mais vendidos e clientes de maior valor.

Comportamento de compras ao longo do tempo.

Objetivos de aprendizado
Nestes projetos eu busquei praticar:

Criação e entendimento de modelo relacional (tabelas, chaves primárias e estrangeiras).

Domínio de SELECT, WHERE, GROUP BY, HAVING, ORDER BY.

Uso de JOINs (INNER, LEFT, RIGHT) para combinar múltiplas tabelas.

Funções de agregação (COUNT, SUM, AVG, MIN, MAX).

Subqueries e CTEs para organizar consultas mais complexas.

Criação de views e consultas reutilizáveis para análise.
